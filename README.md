# 智能仓储举升式AGV小车的设计与控制研究

> **学士学位论文** | 齐鲁工业大学 机械学院 | 2026年4月

## 项目概述

本项目完整记录了举升式AGV小车的设计与控制研究，涵盖总体方案、机械结构、电气系统、运动学建模与PID控制仿真验证。系统以双轮差速驱动为核心，配备举升机构，面向智能仓储场景的物料搬运需求。

- **总体方案设计** — 需求分析、技术指标、差速驱动选型、系统架构
- **机械结构设计** — 移动底盘、驱动轮、升降/旋转机构、罩壳、支撑梁强度校核与有限元分析
- **电气系统设计** — STM32F103C8T6主控、SMC80S伺服电机、传感器选型、控制电路设计
- **运动学建模** — 差速驱动底盘运动学方程推导，四种典型轨迹仿真验证
- **PID控制** — 双轮差速PID控制器设计，扰动条件下目标跟踪性能验证

## 论文章节

| 章节 | 内容 |
|------|------|
| 第一章 引言 | 研究背景（电商物流数据）、国内外AGV与路径规划算法综述、研究内容与方法 |
| 第二章 AGV总体设计方案 | 设计需求与技术指标、三种驱动方式对比选型、系统总体架构 |
| 第三章 AGV机械结构设计 | 移动底盘、驱动轮与减振机构、旋转机构、罩壳；Q235A支撑梁有限元分析（应力/位移/应变） |

| 第五章 运动学建模与仿真 | 差速底盘运动学方程推导、MATLAB四轨迹仿真（直线/圆弧/S形/8字形）、Hybrid A*路径规划、红外PID循迹仿真 |
| 第六章 运动控制算法分析 | PID控制原理、双轮差速PID控制器设计（XY位置环+姿态环+轮速内环）、扰动条件下仿真验证 |
| 第七章 结论 | 工作总结、不足与改进方向、应用前景展望 |

## 关键设计参数

### AGV 整车指标
| 参数 | 数值 |
|------|------|
| 最大外形尺寸 | 1 m × 1 m × 1 m |
| 自重 | 200 kg |
| 额定载重 | 25 kg |
| 举升行程 | 0.5 m |
| 最大行驶速度 | 30 m/min |
| 驱动方式 | 双轮差速 |
| 车体外形 | 椭圆形 |

### 仿真模型参数（MATLAB）
| 参数 | 数值 |
|------|------|
| 驱动轮半径 r | 0.065 m |
| 左右轮距 L | 0.32 m |
| 电机时间常数 τ | 0.15 s |
| 轮速 PID | Kp=3.0, Ki=1.2, Kd=0.03 |
| 姿态 PID | Kp=4.0, Ki=0.3, Kd=0.05 |
| 线速度限幅 | ±0.8 m/s |
| 角速度限幅 | ±1.5 rad/s |
| 传感器距离阈值 | 0.5 m |

## 快速开始

### 编译论文
```bash
cd 1_论文/QLULatex/QLUThesisLatexTemplate-master/Thesis
xelatex main.tex
bibtex main
xelatex main.tex
xelatex main.tex
```

### 运行 MATLAB 仿真
```matlab
cd 2_仿真与代码/matlab/差速
build_diffdrive_pid_model
```
脚本会自动构建 Simulink 模型（XY控制器 → 角度控制器 → 分速度模块 → 左右轮模块 → 运动学模块 → 传感器模块），运行 20s 仿真，并生成轨迹图和左右轮作用力比例图。
